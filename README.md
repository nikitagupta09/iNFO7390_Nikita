# 📊 Data Science Projects Portfolio

This repository contains two data science projects demonstrating skills in **Exploratory Data Analysis (EDA), Data Visualization, and Machine Learning** using Python.

---

## 🚀 Projects Overview

### 1️⃣ Titanic Survival Analysis (EDA & Visualization)

* **Objective:** Analyze the Titanic dataset to identify key factors affecting passenger survival.
* **Techniques Used:**

  * Data Cleaning & Preprocessing
  * Exploratory Data Analysis (EDA)
  * Data Visualization
* **Key Insights:**

  * Gender and passenger class significantly influenced survival rates
  * Younger passengers had higher survival probability
  * Socio-economic status played a major role
* **Tools & Libraries:**

  * pandas, numpy
  * seaborn, matplotlib

---

### 2️⃣ Car Price Prediction (Machine Learning)

* **Objective:** Build a machine learning model to predict car prices based on various features.
* **Techniques Used:**

  * Data Preprocessing & Feature Engineering
  * Exploratory Data Analysis (EDA)
  * Regression Modeling
* **Models Applied:**

  * Linear Regression
  * Decision Tree
  * Random Forest
  * Gradient Boosting
* **Evaluation Metrics:**

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * R² Score
* **Tools & Libraries:**

  * pandas, numpy
  * scikit-learn
  * matplotlib, seaborn
